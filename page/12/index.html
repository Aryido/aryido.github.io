      Multiple points 題型簡單介紹

      August 23, 2023 in algorithm

      Multiple points 的操作，經常用在 array 或 linkedList 上，有幾點事情可以在刷題時特別注意:

      1. 指標會把 list 切成幾個部分，特別注意每一部份的定義

      2. list 是否有排序或可以排序

      3. 指標移動是使用快慢指標還是左右指標

      4. 會不會改變原本的 list

      其中第一點，把 array 切成幾個部分，每個部份的定義，是最重要的思想，可以多思考。
