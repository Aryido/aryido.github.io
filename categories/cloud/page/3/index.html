      Overview Fargate

      December 17, 2022 in cloud, aws

      2017年，AWS 推出了 Fargate，這是一個用於部署和管理容器的模式，代表用戶無需管理 EC2 基礎設施。 Fargate 的核心理念就是 Serverless ，讓 ECS 用戶專注於任務和服務定義，而不是管理集群基礎設施，Fargate 可配合用於 ECS 和 EKS。
